# bankFrame

## 介绍

此项目为本人学习 JavaSE 时，基于 JFrame + 原生jdbc 实现的一个简易的银行系统。整理文件时无意翻出，遂上传，留作学习Java的纪念。

## 功能

- 注册 + 登录
- 存款 + 取款 + 转账
- 销户 + 退出
